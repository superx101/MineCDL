  details: 用户可使用相应的构建器，将MCDL转化为其他平台的命令注册代码
- title: 跨平台
  details: node环境外的其他环境可使用JavaScript引擎运行
- title: 模块化
  details: 用户可根据需要继承 MCDL解析器 或 命令解析器 来修改功能且不会对其他模块造成影响。
